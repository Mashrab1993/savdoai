SavdoAI Print (Android helper)
- Open this folder in Android Studio (Giraffe+). JDK 17.
- Gradle wrapper: gradlew.bat (Windows) / ./gradlew (Unix) — jar included under gradle/wrapper/.
- Build debug APK: gradlew.bat assembleDebug
- Set API base URL in app (Prefs) if not using default Railway API.
- App Links: see ../deploy/print-app-link/
